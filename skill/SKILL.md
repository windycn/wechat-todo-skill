# WeChat Todo Skill

## 触发场景

用户请求涉及以下任一场景时触发：
- 回顾微信聊天内容
- 提取聊天中的待办事项
- 总结群聊/单聊的关键信息
- 查找特定时间段的微信对话
- 分析微信聊天中的截止日期

## 交互流程

### 第一步：确认数据库

系统会自动查找微信数据库。如果未找到，会显示详细的数据库准备步骤：

**权限要求**：
- **Windows 用户**：请确保以管理员权限启动支持 skills 的 agent 程序，否则可能无法从微信内存中提取密钥。
- **macOS 用户**：由于系统安全限制（SIP），需要进行以下操作：
  * **步骤1**: 检查微信版本
    - 确保微信版本为 4.x，否则无法解密数据库
  * **步骤2**: 退出微信
    ```bash
    killall WeChat
    ```
  * **步骤3**: Ad-hoc 签名（需要输入密码）
    ```bash
    sudo codesign --force --deep --sign - /Applications/WeChat.app
    ```
  * **步骤4**: 重新打开微信并登录
  * **步骤5**: 自动安装 wechat-decrypt 工具
    - 系统会自动克隆并安装 `https://github.com/ylytdeng/wechat-decrypt` 工具
    - 安装所需依赖包
  * **步骤6**: 提取密钥
    ```bash
    cd ~/.qclaw/workspace/wechat-decrypt && sudo ./find_all_keys_macos
    ```
  * **步骤7**: 自动查找微信数据目录
    - 系统会自动查找新路径：`~/Library/Containers/com.tencent.xinWeChat/Data/Documents/xwechat_files/*/db_storage`
    - 同时兼容旧路径：`~/Library/Containers/com.tencent.xinWeChat/Data/Library/Application Support/com.tencent.xinWeChat/4.0b*/db_storage`
  * **步骤8**: 自动解密数据库
    - 系统会自动运行 wechat-decrypt 工具解密数据库
    - 解密后的数据会存储在技能目录下的 `decrypted` 文件夹中

**提示**：如果电脑端微信缺少数据，请使用微信的手机聊天记录导出功能导出到电脑微信。

### 第二步：确认分析对象

询问用户要分析的对象：

**若用户未指定**，询问：
> "你想分析哪个群或好友的聊天记录？请提供群名、昵称、备注名或微信号。"

**若用户已指定**，直接进入下一步。

### 第三步：处理模糊匹配

如果用户输入匹配到多个结果，必须列举让用户确认：
> "找到多个匹配项，请选择你要分析的对象：
>  1. [群名/昵称] (备注: xxx)
>  2. [群名/昵称] (备注: xxx)
>  请输入编号或名称进行确认。"

### 第四步：确认日期范围

提供预设选项：
> "请选择要分析的时间范围：
>  - 1) 今日
>  - 2) 昨日
>  - 3) 本周
>  - 4) 最近7天
>  - 5) 自定义范围
>  请输入选项编号。"

用户选择"自定义范围"时，进一步询问具体日期。

### 第五步：提供预设方案

可主动推荐快捷方案：
> "我可以提供以下快速分析：
>  - 今日工作汇总：今天内的待办事项 + DDL
>  - 本周回顾：本週完整摘要 + 关键事件
>  - 待办提取：所有待办事项
>  需要使用预设方案吗？"

## 输出格式

分析完成后，输出结构化报告：

```
## 聊天摘要
- 会话对象: [群名/好友名称]
- 时间范围: [日期起止]
- 消息总数: [N条]
- 分析结果: [主要讨论内容]

## 待办事项
1. [待办内容] (来源: [发送者] @ [时间])
2. [待办内容] (来源: [发送者] @ [时间])
...

## 截止日期 (DDL)
1. [时间点] - [相关上下文]
2. ...

## 关键事件
1. [事件描述] (@ [时间])
2. ...
```

若无待办/DDL/事件，明确告知"未提取到相关事项"。

## 技术规范

- 数据库必须已解密
- 联系人匹配优先级：精确匹配备注名 > 精确匹配昵称 > 精确匹配微信号 > 模糊匹配
- 时间范围使用中国时区 (UTC+8)
- 所有数据处理在本地完成，不涉及网络传输
- 支持增量分析，避免重复处理数据
- 高性能分析，处理速度可达350万条/秒
- 三级缓存机制，缓存查询速度可达1959万次/秒
- 自动数据库查找和解密，减少用户交互
- 支持Windows平台自动解密，macOS平台提供详细的手动解密步骤

## 处理规则

1. **用户未指定分析对象** → 询问确认
2. **匹配到多个结果** → 列举让用户选择
3. **数据库路径无效** → 提示错误并引导重新输入
4. **日期范围不明确** → 使用"今日"作为默认
5. **用户有其他需求** → 遵循用户意愿调整分析范围
6. **电脑端微信缺少数据** → 提醒用户使用手机聊天记录导出功能
